# Student Grade Calculator


## Project Overview
A simple command-line program that accepts a student's name and marks (0-100), validates input, computes a letter grade using if-elif-else logic, and prints an encouraging message. The program includes input validation using a while loop and at least one function for grading logic.


## Grading Rules
- A: 90 - 100
- B: 80 - 89
- C: 70 - 79
- D: 60 - 69
- F: 0 - 59


## Files
- `grade_calculator.py` — Main program
- `test_cases.txt` — Sample test cases
- `screenshots/` — screenshots
